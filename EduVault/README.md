# EduVault MVP

EduVault is an AI-powered academic knowledge platform that allows students to upload, organize, and access study materials. It features a modern, clean Notion-like React frontend and a robust Spring Boot backend that integrates with the Google Gemini AI to analyze study materials.

## Features

- **Centralized Repository:** Browse subjects and notes across semesters.
- **Note Upload:** Upload PDF, PPTX study materials directly to the platform.
- **AI AI Topic Extraction:** Automatically extracts key exam topics from uploaded documents using Gemini 2.5.
- **AI Question Generation:** Automatically generates practice questions based on material context.
- **AI Tutor Chat:** Ask an AI tutor questions specific to your notes or across all uploaded materials.

## Tech Stack

- **Frontend:** React 18, Vite, Tailwind CSS, Lucide React
- **Backend:** Java 17, Spring Boot 3.2, Jackson, Apache PDFBox
- **Database:** Local JSON File Storage (`~/eduvault-data/files/subjects.json`, `notes.json`)
- **AI Integration:** Google Gemini API

## Setup Instructions

### Prerequisites
- Node.js (v18+)
- Java 17 (JDK)
- Maven
- A Google Gemini API Key. Get one from Google AI Studio.

### 1. Start the Backend

Open a terminal and set your Gemini API Key as an environment variable before starting the server.

**Windows (PowerShell):**
```powershell
$env:GEMINI_API_KEY="your_api_key_here"
cd backend
mvn spring-boot:run
```

**Mac/Linux:**
```bash
export GEMINI_API_KEY="your_api_key_here"
cd backend
mvn spring-boot:run
```

*The backend will run on `http://localhost:8080`. It will automatically create the database files in your user home directory under `~/eduvault-data/files/`.*

### 2. Start the Frontend

Open a new terminal session.

```bash
cd frontend
npm install
npm run dev
```

*The frontend will run on `http://localhost:5173`.*

## Seed Data

The backend is configured to automatically seed the `subjects.json` database upon first launch if the file doesn't exist.

To add realistic mock PDF data for AI extraction:
1. Open the application at `http://localhost:5173`
2. Click the "Upload" button
3. Upload any small PDF containing text (e.g., a Wikipedia printout of "Operating Systems")
4. Once processed, click the note in the feed to view AI insights generated by Gemini.

## API Endpoints Overview

- `GET /api/subjects`: Get list of subjects
- `GET /api/notes`: Get notes with optional filters (`subjectId`, `semester`, `keyword`)
- `POST /api/notes/upload`: Multipart upload a document
- `POST /api/ai/chat`: Send a question to the AI tutor with JSON payload `{"question": "What is OS?", "noteId": "optional-id"}`
